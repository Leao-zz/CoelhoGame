WILD SPRITESHEET PACKAGE
========================

Conteúdo:
- wild_atlas_1.png
- wild_atlas_1.json
- wild_atlas_2.png
- wild_atlas_2.json
- wild_animation.json
- pixijs-example.js
- phaser3-example.js

Configuração:
- Total de frames: 240
- Tamanho de cada frame: 300x344px
- FPS sugerido: 30
- Loop: infinito
- Spritesheets: 2
- Tamanho máximo de cada spritesheet: 4096x4096px
- Fundo: preservado conforme os PNGs de origem

Observação:
O JSON usa o formato TexturePacker JSON Hash, compatível com PixiJS e
facilmente adaptável ao Phaser 3. O arquivo wild_animation.json contém
a ordem global dos 240 frames.
